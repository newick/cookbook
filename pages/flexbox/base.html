display: flex

les parents ont le pouvoir !

la propriété display: flex permet d'indiquer au parent qu'il va pouvoir donner des ordres aux enfants.
De manière native (de base donc) display: flex donne déja des ordres.
Elles seront explicitées à chaques propriétés.
Cela explique d'ailleur la différence entre un display: block

  • block
  • block
  • block

et un display: flex

  • row
  • row
  • row

De manière native display: flex donne une direction qui est row et ordonne/impose aux enfants de se mettre en ligne
