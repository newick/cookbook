un modèle simple

Flexbox (pour flexible box) est un mode de mise en page prévoyant une disposition des éléments d’une page de telle sorte que ces éléments possèdent un comportement prévisible lorsqu’ils doivent s’accommoder de différentes tailles d’écrans/appareils (src mdn).

Voici son implantation dans les navigateurs via caniuse
Mon sentiment sur le modèle que propose flexbox est qu'il apparait simple, ludique et très accessible (surtout à son premier niveau).
La pratique peut faire ressortir quelques résultats étranges mais la courbe d'apprentissage, un outillage intélligent et le fun procuré font de flexbox une découverte cool en CSS.

Pour utiliser un language commun et vous permettre d'explorer correctement je vous propose
de raisonner parent/enfant, le plus sémantique est donc un modèle <ul> / <li>
Le parent est donc le conteneur des enfants, il les englobe.

<ul> est donc le parent et les <li> sont ses enfants.
Le parent va passer un ensemble d'ordre aux enfants.
Les enfants ont la capacité d'avoir des comportements qui leur sont propres.

J'aime à dire sous forme de métaphore caricaturale:
C'est comme dans la vrai vie, le parent ordonne, l'enfant s'execute, l'adolescent se rebelle.

    proposition de déroulé

    Flexbox est simple, nous esperons que son déroulé le sera tout autant.
    les ojectifs sont:

    • Découvrir une propriété CSS d'une manière abordable par tous
    • Simplifier le positionnement des élements d'une interface en CSS
    • Avoir une solution alternative aux autres techniques de positionnement
