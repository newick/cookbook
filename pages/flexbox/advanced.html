flex-grow

l'espace positif

Qu'est ce que l'espace positif ?
l'espace positif est l'espace qui reste disponible dans le conteneur, en largeur si row, en hauteur si column

  • aucun effet
  • aucun effet
  • aucun effet

Grow travaille sur cet espace positif, il va le prendre puis le répartir parmi tout les enfants, d'où le fait que grow agrandisse vos enfants
Il va faire selon selon un indice et par default grox = 0

Cela explique pourquoi si un seul des enfants porte un grow ave une valeur de 1 ou +, cela ne change rien.
C'est bien lui qui récupère tout l'espace diponible, les autres enfants étant avec un indice de 0

flex-shrink

l'espace négatif

Shrink est l'opposé de Grow, il travaille avec l'espace négatif.
L'espace négatif est l'espace qui va se réduire lorsque vous allez redimentionner la page (selon l'axe de travail (row ou column).
l'espace réduisant va se répartir sur les enfant selon un indice qui de base est à 0

order

une question de poid

order permet à un enfant de prendre la place qu'il souhaite grâce à un indice
Par défault cet indice n'a pas de valeur
A l'origine nous avons donc dans l'ordre
